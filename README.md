This project uses Machine Learning (ElasticNet Regression) to predict the monthly closing price of Yes Bank stock based on Open, High, Low, and Previous Close data.

## Features
- **Interactive Web App:** Built with Streamlit.
- **Model:** Tuned ElasticNet Regression.
- **Data:** Historical Yes Bank stock prices.

## How to use
Adjust the sliders to set the market parameters and click 'Predict' to see the estimated closing price.
